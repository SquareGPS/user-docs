# No movement

Reports when a device that is expected to move stays stationary. It is configured on the device and reports an event to the Navixy platform.

## Settings

* **Enable**: turn no-movement detection on or off.
* **Timeout**: how long the device may stay stationary before an event is reported.
* **Pre-alarm duration**: an optional warning period before the main event.

## Availability

Appears on device models that support no-movement detection.
