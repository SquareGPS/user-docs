# SMS channel usage

## Purpose

The **SMS channel usage** block controls how the device uses its SMS channel, for example to send alerts when the battery is low.

## Settings

* **Low-battery mode**: control whether the device sends SMS alerts on low battery.

## Appears when

Appears on device models that expose SMS channel options.
