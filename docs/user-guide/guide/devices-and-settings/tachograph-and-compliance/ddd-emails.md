# DDD files recipient email

Automatically sends downloaded DDD files to the people who need them, for archiving, compliance record-keeping, or forwarding to an analysis service. It pairs with [DDD download](ddd-download.md): once a file is retrieved from the device, it's emailed to the configured recipients.

<!-- SCREENSHOT: DDD files recipient email block, list of email fields with add (+) button. Annotate: add/remove recipient, the 5-recipient limit. -->

## Settings

* **Recipient emails**: Add the email addresses that should receive downloaded DDD files. Click the **+** button to add each address. You can specify up to 5 recipients.

## Availability

Appears on tachograph-capable devices that support DDD download.

## Limitations

* Up to 5 email addresses can be configured.
* DDD files are emailed after a download completes: Configure download separately on the [DDD download](ddd-download.md) page.
* Navixy only delivers the files. It does not read or process their contents.

## See also

* [DDD download](ddd-download.md): request and download DDD files from the device.
