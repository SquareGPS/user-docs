# Object management

These blocks control a device's **identity and administration** — how it's named and organized in your fleet, its SIM and trusted numbers, its plan, and tools for working statuses, raw data export, and time zone.

<table data-view="cards"><thead><tr><th>Block</th><th>Purpose</th><th data-card-target data-type="content-ref">Target</th></tr></thead><tbody><tr><td><strong>Object label</strong></td><td>Name the device and organize it with a group and tags.</td><td><a href="object-label-block.md">Object label block</a></td></tr><tr><td><strong>Phone number</strong></td><td>SIM phone number and mobile-data (APN) settings.</td><td><a href="phone-number-block.md">Phone number block</a></td></tr><tr><td><strong>Trusted phone numbers</strong></td><td>Numbers allowed to send SMS/USSD commands to the device.</td><td><a href="trusted-phone-numbers.md">Trusted phone numbers</a></td></tr><tr><td><strong>Change plan</strong></td><td>Assign the device's pricing plan (tariff).</td><td><a href="change-plan-block.md">Change plan block</a></td></tr><tr><td><strong>Working statuses</strong></td><td>Attach selectable work states (e.g. Available, In service).</td><td><a href="working-statuses-block.md">Working statuses block</a></td></tr><tr><td><strong>Raw data</strong></td><td>Export the device's raw telemetry as a CSV file.</td><td><a href="raw-data-block.md">Raw data block</a></td></tr><tr><td><strong>Device time zone</strong></td><td>Set the device's time-zone offset.</td><td><a href="device-timezone-block.md">Device timezone block</a></td></tr></tbody></table>
