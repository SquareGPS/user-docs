# Data retention period (increase from 1 year)

## Question

Is it possible to increase my data limit from 1 year?

## Answer

By default, your platform and associated device plans allow up to the last **one year** of data to be displayed on the platform.

However, you can update this to up to **three years** by working with your Customer Success Manager.

This data, even when not displayed, is still saved for up to three years. If you need it after the fact, it can be provided on request.
