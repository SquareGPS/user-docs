# Add custom links inside the platform (User Applications)

### Question

Can I add my own links to the platform?

### Answer

Yes.

Use **User Applications**.

It adds a sidebar button opening your URL in an iframe or new tab.

You can also pass the user hash to the target, which is useful if you want to integrate with Navixy API from your own app.

For iframes, the link must be HTTPS.

{% hint style="info" %}
Support does not troubleshoot third-party iframe issues (for example, if your site blocks iframe embedding).
{% endhint %}
