# Clone a device across users

## Question

How can I display a device across multiple users?

## Answer

It's possible to create a clone of a device between users in Admin Panel.

Go to the **Trackers** section, select the tracker, and use the **Clone** option.

Clones don't come at an additional charge as they aren't a real device, and you can have as many as you would like.

If you need to clone a device across panels (for example, between two separate resellers), use the Admin Panel API clone method or contact Support.
