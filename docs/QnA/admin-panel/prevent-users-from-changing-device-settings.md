# Prevent users from changing device settings

### Question

How can I prevent users from changing device settings?

### Answer

In order to prevent users from changing device settings (or even disallowing them from registering a new device), you need the **Menu Editor** functionality.

With Menu Editor, you can remove menu items such as **Devices and Settings**.

This option requires both Fleet Management and Field Service add-ons. If you don’t have this functionality, contact your Customer Success Manager to enable it.
