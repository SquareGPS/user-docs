# Zero saving value in fuel reports

### Question

Why does the saving/overrun fuel value show zero in my fuel report?

![](<../.gitbook/assets/Unknown image (118)>)

### Answer

In this column, the difference is shown between the standard fuel consumption and the actual consumption calculated in the report. The standard consumption is taken from the vehicle to which the tracker is linked. If there is no such link or the standard consumption is not specified, this report column will always be empty (zero value).
