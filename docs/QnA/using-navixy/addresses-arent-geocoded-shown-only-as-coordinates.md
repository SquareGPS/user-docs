# Addresses are not geocoding (coordinates only)

### Question

My addresses are showing as only coordinates, what’s going on?

### Answer

If your device locations and reports are showing coordinates only and not the associated address, the most likely culprit is that the system has reached the geocoding limit set on the system. This is taxing on the system, so a limit was put in place to prevent it from being overloaded.

Regarding the limits themselves, our Basic GIS package allows for up to 100,000 geocoding calls a day per user and our Premium GIS packages allow for up to 500,000 calls a day per user. This limit is reset at 00:00 UTC+0. If you wish to increase this limit, please reach out to your Customer Success Manager.
