# Filter forms based on images

## Question

Why is it not possible to filter forms based on images?

## Answer

Unfortunately, in the form's submission, it isn't possible to filter entries based on images. In fact, the platform doesn't allow filtering by every column in the form. This is because the filtering logic is limited to parameters categorized as **business data**, such as object, addresses, dates, tasks, IDs, and among other fields.
