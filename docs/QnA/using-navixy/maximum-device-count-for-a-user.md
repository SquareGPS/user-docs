# Maximum device count for a user

### Question

What is the maximum device count for a user?

### Answer

Currently, the maximum device count is 100,000 devices per user. Please note, however, that viewing all 100,000 devices won’t function due to technical difficulties, so in order to prevent technical slow downs, we have implemented a hard display limit of 25,000 devices and our platform enforces clustering of devices on the map after showing 300+ devices at once.
