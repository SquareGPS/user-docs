# Maximum device count for a user

### Question

What is the maximum device count for a user?

### Answer

Currently, the global device limit is 2,000,000 devices per user, though this number can be restricted by your tariff. However, viewing all devices isn't possible due to technical difficulties. In order to prevent technical slowdowns, we implemented a hard display limit of 25,000 devices, and our platform enforces clustering of devices on the map after showing 300+ devices at once.
