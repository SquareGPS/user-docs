# Navixy SMS gateway

## Question

Can Navixy provide SMS gateway service to clients?

## Answer

Yes, but only in limited cases.

Navixy can provide an SMS gateway:

* for a limited period of time
* with a limited number of messages
* only when the Customer Success Manager provides a valid justification

Support sets it up and disables it after the agreed period ends.
