# Adjust incoming data with IoT Logic

### Question

Can I adjust incoming data from a device to suit my needs?

### Answer

Yes, if the data sent by your device isn’t quite what you need, you’re able to adjust it by utilizing our IoT Logic functionality. This option allows you to convert data how you need by using mathematical functions to adjust, change, and combine data with the Initiate Attribute node.

We recommend reviewing our IoT Logic documentation for further information.
