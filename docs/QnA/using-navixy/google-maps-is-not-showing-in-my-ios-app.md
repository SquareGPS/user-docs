# Google Maps is not showing in my iOS app

## Question

Why doesn’t Google Maps appear in the iOS app?

## Answer

The iOS version of **X-GPS Monitor** uses Apple Maps by default and doesn't support Google Maps. However, the iOS version of **X-GPS Mobile** does offer this option starting with v1.5.0. Update your app!
