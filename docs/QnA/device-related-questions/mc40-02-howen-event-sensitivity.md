# MC40-02 Howen event sensitivity

## Question

My camera generates too many false events. Can I change sensitivity?

## Answer

Sensitivity cannot be adjusted through Navixy.

Use **Howen iTools**.

To change sensitivity, log in to the camera via Howen iTools, select either DMS or ADAS events, and then click the specific event you want to configure. There you can adjust:

* sensitivity (L/M/H)
* speed
* trigger hold
* interval
* buzzer
* other event-related parameters
