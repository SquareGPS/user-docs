# On-premise updates FAQ

### How often does my platform get updated?

Navixy platform does not have automatic update delivery functionality. All updates are performed only on the owner's initiative. The number of updates is not limited, you can do this procedure at any suitable time.

### How often are new versions released?

We don't have a strict release cycle. Major builds of the platform containing significant changes and bringing new functionality are released every 2-4 months. Minor versions, bringing mainly fixes and improvements to the current functionality, are released up to several times a month.

### Do I need to install each new update?

We recommend keeping the platform updated to receive all updates and fixes in time.

However, some of our clients prefer to update less frequently and settle for basic functionality. The decision is yours. An unupdated version of the platform does not lose its functionality.
