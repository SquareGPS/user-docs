# What operation system to choose?

### What operating systems are supported?

Navixy On-premise solution is fully cross-platform. That is, it can be deployed on any up-to-date operation sysem of the Linux family, as well as on a Windows server.

### Which operation system to choose?

You are free to choose any OS that your technical team has sufficient experience with. We generally recommend the latest version of **Ubuntu**, as it provides maximum ease of maintenance and shows itself to be stable in long-term use. We also feature the advanced Navixy platform installation scripts available only for Ubuntu.

### Do you offer tools for operating system administration?

No, we only carry out development of Navixy software. Since it is cross-platform, it does not depend on OS type and version, so it does not require special system presetting. All OS maintenance is performed by the client's technical specialists.
