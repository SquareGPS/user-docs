# Reading data from an Eye Sensor on the platform

### Question

How can I read data from an eye sensor on the Navixy platform?

### Answer

Once we have confirmed information through Teltonika´s configurator, then it will be necessary to go to Device Settings, select the device linked to the Eye Sensor, and create a measurement sensor selecting the AVL input where we are reading the humidity, temperature, magnetism or speed information.
