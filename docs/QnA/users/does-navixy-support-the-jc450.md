# Does Navixy support the JC450?

### Question

Does Navixy support the JC450?

### Answer

Yes, Navixy supports the JC450 through Lightmetrics. Please note that the JC450 is not natively integrated into Navixy; support is provided via a third-party partnership.

Navixy can retrieve telematics data, while the video is stored and managed in Lightmetrics. According to the latest updates, the camera may operate with some limitations, so we recommend contacting Lightmetrics directly to confirm current camera performance and any constraints.
