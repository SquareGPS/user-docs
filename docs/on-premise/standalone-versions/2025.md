# 2025

## 2025.08 (24 Oct 2025)

In build 2025.08, we continued to add 3rd-party solutions for video telematics, including device-agnostic ones. Now you can choose a video telematics provider that fits your business requirements.

In our relentless pursuit of quality, we continue working with customer feedback and providing issue fixes and platform improvements that are continuously added to our standalones.&#x20;

<summary>Integrated</summary>

* [LightMetrics ](https://marketplace.navixy.com/shop/lightmetrics-video-telematics/)- a video telematics platform supporting devices from multiple manufacturers. It can now be embedded into Navixy UI to combine telematics data and video monitoring in the same workspace.

## 2025.05 (25 Jul 2025) <a href="#id-25-jul-2025-2025.05.295" id="id-25-jul-2025-2025.05.295"></a>

In this standalone version 2025.05, we've integrated not only a significant number of devices requested by clients, but also introduced the first 3rd-party solution for video telematics, which broadens the horizon of video monitoring without switching from Navixy UI.

We also continue working on the quality. This build facilitates the installation and update processes by providing user-friendly scripts and contains several bugfixes.

<summary>Improved</summary>

* Enabled [IoT Logic](https://www.navixy.com/iot-logic/) functionality for standalones, available on demand.
* Deployment - Dockered standalone evolves as the easiest platform deployment method, introducing [a script for initial configuration](https://www.navixy.com/docs/on-premise/on-premise/how-to-guide/installation/easy-installation#step-2-deploy-navixy-platform) in the form of a minimalistic graphic wizard and an easy-to-use [update script](https://www.navixy.com/docs/on-premise/on-premise/how-to-guide/installation/update/update-linux/automatic-update-linux#automated-update).
* Upgraded the platform runtime from Java 17 to Java 21 (LTS).
* Fixed the issue causing the inability to choose analog input 2 for [SAT-LITE 4 (SLXT/SLXU)](https://www.navixy.com/devices/satellite-solutions/sat-lite-4-slxt-slxu/)​
* Fixed the issues with adding and configuring sensors for [TOPFLYTECH Torchx 110](https://www.navixy.com/devices/topflytech/topflytech-torchx-110/).
* Fixed the issue with parsing battery percentage data from [Queclink GL33CG](https://www.navixy.com/devices/queclink/queclink-gl33cg/).
* Minor DB updates to improve software performance.
