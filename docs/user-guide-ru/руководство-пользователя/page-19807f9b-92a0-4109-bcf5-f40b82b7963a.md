# Аккаунт

Добро пожаловать в раздел документации по учетной записи пользователя. В этом руководстве содержится исчерпывающая информация об управлении учетной записью пользователя Navixy и настройке параметров в соответствии с вашими потребностями.

Изучите каждый раздел меню, чтобы максимально использовать преимущества вашей учетной записи Navixy. Вот некоторые из наиболее часто используемых разделов:

- [**Основные параметры пользователей**](./page-36c7764b-812b-468f-9672-2c25e3558ae9/page-06f03de4-e453-43d1-9034-4cc7091142d6.md)  
Узнайте, как настроить параметры учетной записи, включая языковые предпочтения, системы измерения, часовые пояса и форматы даты.
- [**Пользователи и роли**](./page-c513bb6a-9d49-4b06-a83a-75c333d18f64/page-7632d1e0-f55d-4982-b086-2983a13106f6.md)  
Управляйте своей учетной записью, добавляя новых пользователей, определяя роли пользователей для управления доступом и разрешениями, а также настраивая контроль доступа, чтобы пользователи видели только информацию, соответствующую их ролям.
- [**Восстановить пароль**](./page-b0878627-26b0-481e-927d-25c60e151841/page-4ffdbc0b-865c-4718-80b5-54507fec8b31.md)  
Пошаговое руководство по восстановлению пароля, если вы его забыли.
- [**Ключи API**](./page-26dd40c8-4cba-42f1-9fb7-5bb1d3e84d65/api.md)  
Генерируйте и управляйте ключами API для интеграции сторонних приложений с вашей учетной записью.
